# Waqooyi Bari Team Website

Premium website for Waqooyi Bari Team — a reading and writing community.

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript**
- **TailwindCSS**
- **Framer Motion** — animations
- **GSAP** — advanced animations (ready to integrate)
- **Lenis** — smooth scrolling

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

| Route | Description |
|-------|-------------|
| `/` | Home page with all sections |
| `/about` | About the team |
| `/leadership` | Team leadership |
| `/members` | Team members |
| `/books` | Books library with search & filters |
| `/events` | Events timeline |
| `/gallery` | Photo gallery |
| `/news` | News & articles |
| `/support` | Donations & sponsors |
| `/contact` | Contact information |
| `/admin` | Admin dashboard |

## Design

- Navy blue (`#07152E`) and gold (`#D4AF37`) color scheme
- Glassmorphism cards
- Framer Motion animations
- Fully responsive
